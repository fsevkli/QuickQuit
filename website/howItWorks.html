How It Works

A button is added to your website so the user can activate the extension. If the user decides not to download the extension, they will still be able to leave the website."

How to add the button to your website

  1. Please enter any domain names associated with your site. Any page under this domain will be removed from the users history.
  2. Please choose what you would like the button text to say.
  3. Please choose what kind of auto-generated content will replace the user's browser history.
    • Dropdown item text
    • News articles
    • YouTube videos
    • Something else here
  4. Copy and paste this code into your website's HTML header.
